# Immutable X

**Immutable X** 是專為 NFT 和遊戲設計的 Ethereum 第二層擴容方案（Layer 2），提供零 gas 費用的 NFT 鑄造和交易體驗。

| 項目 | 詳情 |
|------|------|
| **成立時間** | 2018年 |
| **創辦人** | Robbie Ferguson、James Ferguson |
| **總部** | 澳洲悉尼 |
| **技術** | StarkEx 零知識證明卷軸 |
| **代幣** | IMX |
| **Animoca 角色** | 重要投資者和生態夥伴 |

---

## 技術架構

### StarkEx 零知識證明
Immutable X 基於 StarkWare 的 StarkEx 技術：
- **有效性證明**：使用 STARK 技術確保交易有效性
- **鏈下執行**：計算在鏈下進行，僅在鏈上驗證證明
- **繼承安全性**：保持 Ethereum 主網的安全級別

### 核心優勢
| 特性 | 說明 |
|------|------|
| 零 Gas 費用 | 鑄造和交易 NFT 無需支付 gas |
| 即時確定性 | 交易立即確認，無需等待區塊確認 |
| 高吞吐量 | 每秒可處理超過 9,000 筆交易 |
| 碳中和 | 高效能設計，無需大量能源消耗 |

## 遊戲生態系統

### 知名遊戲項目
Immutable X 是 Web3 遊戲的首選平台之一：

- **Gods Unchained**：數字卡牌遊戲，由 Immutable 自研
- **Guild of Guardians**：手機 RPG 遊戲
- **Illuvium**：開放世界 RPG 和自動戰鬥遊戲
- **Ember Sword**：沙盒 MMORPG
- **Star Atlas**：太空探索元宇宙

### 開發者工具
- **Immutable SDK**：多平台開發工具包
- **市場 API**：輕鬆整合 NFT 交易功能
- **錢包整合**：支援多種錢包解決方案

## IMX 代幣

### 代幣用途
| 功能 | 說明 |
|------|------|
| 交易費用 | 支付協議費用（以 IMX 計價，可選） |
| 質押 | 質押 IMX 獲得獎勵和治理權 |
| 治理 | 參與協議升級和資金分配投票 |

### 代幣經濟
- **總供應量**：20 億枚
- **分配**：生態基金、團隊、私募、公開銷售
- **質押獎勵**：協議手續費的一部分分配給質押者
