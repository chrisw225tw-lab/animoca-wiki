# Animoca Brands 公司概況

## 公司簡介

**Animoca Brands Corporation Limited** 是一家總部位於香港的數字資產領導企業，專注於區塊鏈和代幣化資產，推動 Web3 創新的未來。

| 項目 | 詳情 |
|------|------|
| **成立時間** | 2014年（最初為遊戲開發商） |
| **總部** | 香港 |
| **創辦人** | Yat Siu（蕭逸） |
| **核心使命** | 推動數字產權（Digital Ownership），建立開放的元宇宙 |
| **2026年戰略** | 穩定幣、RWA（真實世界資產）、AI、DePIN |

## 2026 年重大事件：納斯達克上市

Animoca Brands 正通過與 **Currenc Group（NASDAQ: CURR）** 的反向併購，計劃在 **2026年第三季度** 於納斯達克上市。

| 項目 | 詳情 |
|------|------|
| **交易結構** | 反向併購（Reverse Merger） |
| **預計完成** | 2026年 Q3 |
