# Polygon

**Polygon**（前身為 Matic Network）是 Ethereum 的領先擴容解決方案，提供多種工具幫助開發者構建可擴展的去中心化應用。

| 項目 | 詳情 |
|------|------|
| **成立時間** | 2017年 |
| **創辦人** | Jaynti Kanani、Sandeep Nailwal、Anurag Arjun |
| **總部** | 印度班加羅爾 |
| **代幣** | MATIC（2024 年升級為 POL） |
| **技術** | PoS 側鏈、ZK Rollups |
| **Animoca 角色** | 戰略投資者和生態夥伴 |

---

## 技術架構

### 多層擴容方案
Polygon 提供完整的擴容工具套件：

| 解決方案 | 類型 | 特點 |
|----------|------|------|
| Polygon PoS | 側鏈 | EVM 兼容，低手續費 |
| Polygon zkEVM | ZK Rollup | Ethereum 級安全性 |
| Polygon Miden | ZK Rollup | STARK 技術 |
| Polygon CDK | 開發工具 | 自定義 ZK 鏈 |

### Polygon PoS 鏈
- **EVM 兼容**：Ethereum 應用無縫遷移
- **低手續費**：交易費用低於 $0.01
- **快速確認**：約 2 秒區塊時間
- **高吞吐量**：每秒 7,000+ 交易

## MATIC / POL 代幣

### 代幣功能
| 用途 | 說明 |
|------|------|
| Gas 費用 | 支付 Polygon 網絡交易費 |
| 質押 | 驗證者質押以保障網絡安全 |
| 治理 | 參與協議升級投票 |
| 抵押 | DeFi 應用中的抵押資產 |

### 代幣升級（MATIC → POL）
2024 年，MATIC 升級為 POL：
- **多鏈質押**：一個代幣可質押在多個 Polygon 鏈
- **增強功能**：支持更多擴容方案
- **無縫遷移**：1:1 兌換，無需操作

---

## Polygon CDK（鏈開發套件）

### 概念
Polygon CDK（Chain Development Kit）讓開發者能輕鬆創建自己的 ZK 驅動 Layer 2：

- **模組化設計**：選擇所需的組件和配置
- **ZK 技術**：內建零知識證明支援
- **即時終局性**：快速交易確認
- **統一流動性**：與 Polygon 生態共享流動性

### 採用項目
- **Immutable**：遊戲專用 ZK 鏈
- **Astar Network**：日本企業級鏈
- **IDEX**：去中心化交易所
- **Nexo**：加密借貸平台
