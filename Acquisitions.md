## 2026年收購

### Somo（2026年 1月）
| 項目 | 詳情 |
|------|------|
| **目標公司** | Somo |
| **收購時間** | 2026年 1月 |
| **業務領域** | 數字資產和區塊鏈服務 |
| **戰略意義** | 強化數字資產諮詢能力 |

## 收購策略分析

### 戰略目標
| 目標 | 說明 |
|------|------|
| **人才獲取** | 收購優秀的遊戲開發團隊 |
| **IP 儲備** | 獲取有潛力的遊戲 IP |
| **技術能力** | 增強區塊鏈整合能力 |
| **用戶基礎** | 快速獲取現有用戶群 |

### 整合模式
- **保留品牌**：被收購公司通常保留原有品牌
- **資源注入**：提供資金和區塊鏈專業知識
- **協同效應**：整合到 Animoca 生態系統
- **獨立運營**：給予被收購團隊創作自由
