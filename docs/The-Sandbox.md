# The Sandbox

**The Sandbox** 是 Animoca Brands 旗下的旗艦元宇宙平台，基於區塊鏈的虛擬世界。

| 項目 | 詳情 |
|------|------|
| **類型** | 元宇宙（Metaverse）平台 |
| **代幣** | $SAND |
| **核心** | UGC（用戶生成內容）+ 遊戲 |
| **特色** | 虛擬土地（LAND）、NFT 資產 |

---

## 核心要素

### 1. LAND（虛擬土地）
- 用戶可購買、擁有、開發虛擬土地
- 土地是 NFT，可交易
- 2026年土地銷售持續增長

### 2. ASSETS（資產）
- 用戶創建的 3D NFT 資產
- 可用於遊戲、裝飾、交易

### 3. Game Maker
- 無需編程即可創建遊戲
- UGC 生態的核心工具

### 4. VoxEdit
- 3D 建模工具
- 創建體素風格資產
