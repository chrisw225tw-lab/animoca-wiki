# Moca Chain

**Moca Chain** 是專為數字身份和用戶數據設計的 Layer 1 區塊鏈，為 [[Moca-Network]] 生態提供底層基礎設施支持。

| 項目 | 詳情 |
|------|------|
| **類型** | Layer 1 區塊鏈 |
| **定位** | 身份與用戶數據專用鏈 |
| **所屬生態** | [[Moca-Network]] |
| **開發商** | [[Animoca-Brands]] |

---

## 核心特點

### 1. 身份優先設計
- 原生支持數字身份協議
- 用戶數據主權歸屬
- 可驗證憑證（Verifiable Credentials）支持

### 2. 高性能架構
- 高吞吐量交易處理
- 低 Gas 費用設計
- 快速交易確認時間

### 3. 隱私保護
- 零知識證明原生整合
- 選擇性數據披露
- 符合隱私法規要求

---

## 技術架構

| 層級 | 功能 |
|------|------|
| **應用層** | [[Moca-ID]]、[[Moca-Proof]]、[[Moca-Portfolio]] |
| **協議層** | 身份協議、數據管理 |
| **共識層** | 安全高效的共識機制 |
| **網絡層** | P2P 網絡傳輸 |

---

## 生態應用

Moca Chain 支持以下核心應用：

- **[[Moca-ID]]** — 跨鏈數字身份管理
- **[[Moca-Proof]]** — 身份驗證與獎勵
- **[[Moca-Portfolio]]** — 資產配置平台
- **第三方整合** — 開放 API 供合作夥伴使用

---

## 互操作性

Moca Chain 設計為多鏈互通：
- 與 Ethereum 跨鏈橋接
- Solana 生態整合
- Cosmos IBC 協議支持
- 未來將支持更多主流公鏈
