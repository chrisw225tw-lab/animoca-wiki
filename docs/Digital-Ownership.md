# Digital Ownership

**Digital Ownership**（數字產權）是 [[Animoca-Brands]] 的核心哲學，指用戶真正擁有和控制自己的數字資產，而非僅有使用許可。

---

## 傳統 vs Web3 所有權

### Web2 的虛假所有權
| 場景 | 實際情況 |
|------|----------|
| 遊戲道具 | 遊戲公司可隨時收回 |
| 音樂購買 | 僅獲得播放許可，無所有權 |
| 社交媒體帳號 | 平台可無理由封禁 |
| 電子書 | 無法轉售或贈送 |

### Web3 的真正所有權
- **資產存儲在區塊鏈上**，而非公司伺服器
- **私鑰證明所有權**，無需平台許可
- **自由轉讓和交易**，真正的資產流動性
- **跨平台使用**，資產不綁定單一應用

---

## 數字產權的核心要素

### 1. 可驗證的所有權
- 區塊鏈作為公共帳本
- 交易歷史透明可查
- 無法偽造或篡改

### 2. 真正的控制權
| 權利 | 說明 |
|------|------|
| 轉讓權 | 可出售、贈送或繼承 |
| 使用權 | 在支援的平台自由使用 |
| 收益權 | 從資產獲得收益（如出租）|
| 處分權 | 銷毀或修改資產 |

### 3. 永久性
- 只要區塊鏈存在，資產就存在
- 不受公司倒閉或服務關閉影響
- 真正屬於用戶的資產

---

## 可互操作性 (Interoperability)

### 概念
可互操作性指數字資產可以在不同的應用和平台之間使用。

### 例子
- **遊戲道具**：在遊戲 A 獲得的劍，可在遊戲 B 使用
- **虛擬時尚**：購買的數字服裝可穿在不同元宇宙平台
- **身份系統**：[[Moca-ID]] 可在多個應用使用

### 技術基礎
- **開放標準**：ERC-721、ERC-1155 等 NFT 標準
- **區塊鏈網絡**：Ethereum、Polygon 等公共鏈
- **跨鏈橋接**：不同區塊鏈間的資產轉移

## Animoca 的數字產權實踐

### The Sandbox
- 用戶擁有 LAND 和 ASSET NFT
- 可自由建造、出租或出售虛擬土地
- 創作者保留內容的所有權

### Mocaverse
- [[Moca-NFT]] 是會員資格的證明
- 持有者獲得生態權益
- 可在二級市場自由交易

### 投資哲學
[[Animoca-Brands]] 投資的所有項目都遵循數字產權原則：
- 用戶擁有遊戲內資產
- 開放的經濟系統
- 反對封閉平台和圍牆花園

## 開放元宇宙 (Open Metaverse)

### 願景
- 不同元宇宙平台之間互通
- 用戶資產可跨平台使用
- 沒有單一公司控制整個虛擬世界

### 與封閉元宇宙的對比
| 開放元宇宙 | 封閉元宇宙 |
|------------|------------|
| 用戶擁有資產 | 平台擁有資產 |
| 可跨平台使用 | 僅限單一平台 |
| 社群治理 | 公司控制 |
| 開放標準 | 專有技術 |
