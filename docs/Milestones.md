# Animoca Brands 里程碑

本頁記錄 [[Animoca-Brands]] 發展歷程中的重要事件和里程碑。

---

## 2014：創立之年

| 時間 | 事件 |
|------|------|
| 2014年 | Yat Siu 在香港創立 Animoca Brands |
| 2014年 | 最初專注於手機遊戲開發和發行 |

**背景**：公司從傳統遊戲業務起步，在亞洲市場建立了初步基礎。

---

## 2018：區塊鏈轉型

| 時間 | 事件 |
|------|------|
| 2018年 | 宣佈全面轉向區塊鏈遊戲 |
| 2018年 | 確立「數字產權」核心使命 |
| 2018年 | 開始投資首批區塊鏈初創公司 |

**意義**：這是公司的重大轉折點，從傳統遊戲轉向區塊鏈和 NFT 領域。

---

## 2019：退市與專注

| 時間 | 事件 |
|------|------|
| 2019年 | 從澳洲證券交易所（ASX）退市 |
| 2019年 | 專注於私營發展，不受公開市場短期壓力 |

**影響**：退市讓公司能夠進行長期投資，不受季度財報壓力。

---

## 2021：元宇宙元年

| 時間 | 事件 |
|------|------|
| 2021年 | [[The-Sandbox]] Alpha 版本發布 |
| 2021年 | NFT 市場爆發，投資組合價值大幅增長 |
| 2021年 | 投資組合擴展至 100+ 公司 |
| 2021年 | 完成多輪大額融資 |

**亮點**：The Sandbox 成為頂級元宇宙平台，Animoca 確立了行業領導地位。

---

## 2022：生態擴張

| 時間 | 事件 |
|------|------|
| 2022年 | Mocaverse NFT 系列發行 |
| 2022年 | 投資組合達到 300+ 公司 |
| 2022年 | 熊市中繼續投資，佈局下一代項目 |
| 2022年 | 估值達到 60 億美元 |

**策略**：在市場低迷期持續投資，為下一輪增長做準備。

---

## 2023：基礎設施建設

| 時間 | 事件 |
|------|------|
| 2023年 | [[Moca-Network]] 正式上線 |
| 2023年 | Moca ID 數字身份系統發布 |
| 2023年 | 投資組合達到 450+ 公司 |
| 2023年 | 強化亞洲市場佈局 |

**重點**：從投資轉向自有基礎設施建設，Moca 生態系統成型。

---

## 2024：投資組合壯大

| 時間 | 事件 |
|------|------|
| 2024年 | 投資組合突破 500 家公司 |
| 2024年 | 可觸達用戶達到 7 億+ |
| 2024年 | 深化穩定幣和 RWA 佈局 |
| 2024年 | 多個被投公司成功退出 |

**成果**：投資策略開始顯現回報，生態協同效應增強。

---

## 2025：戰略轉型

| 時間 | 事件 |
|------|------|
| 2025年 Q1 | 宣佈重大戰略轉型 |
| 2025年 | 四大新重點：穩定幣、RWA、AI、DePIN |
| 2025年 | 與渣打銀行、香港電訊宣佈穩定幣合作 |
| 2025年 | NUVA RWA 平台發布 |
| 2025年 | 宣佈納斯達克上市計劃 |

**轉變**：從純 Web3 遊戲投資者轉型為全面的數字資產綜合企業。

---

## 2026：納斯達克上市

| 時間 | 事件 |
|------|------|
| 2026年 1月 | 收購 Somo 公司 |
| 2026年 Q3 | 預計完成納斯達克上市（通過與 Currenc Group 合併）|
| 2026年 | 估值目標 10 億美元 |

**目標**：成為全球首家在主要交易所上市的數字資產綜合企業。
