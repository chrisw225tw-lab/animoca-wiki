# Realm Points

**Realm Points** 是 Moca ID 生態系統中的忠誠度積分系統，用於獎勵用戶在 Web3 生態中的參與和貢獻。

| 項目 | 詳情 |
|------|------|
| **類型** | 忠誠度積分 / 聲譽積分 |
| **所屬生態** | [[Moca-ID]] |
| **獲取方式** | 完成任務、參與活動、合作夥伴體驗 |
| **用途** | Cosmic Club 消費、等級提升、獨家獎勵 |

---

## 如何賺取 Realm Points

### 1. 完成任務 (Quests)
- 每日登入任務
- 社交媒體互動（Twitter、Discord）
- 鏈上交易活動
- 教育內容學習

### 2. 合作夥伴體驗
- 參與 [[Animoca-Brands]] 投資組合公司的產品
- 試玩新遊戲和應用
- 參加線上線下活動

### 3. 生態貢獻
- 邀請新用戶加入 Moca ID
- 提供產品反饋
- 參與社群治理

---

## Realm Points 用途

### Cosmic Club 消費
- 在 [[Cosmic-Club]] 遊戲平台中使用
- 購買遊戲道具和皮膚
- 參加付費錦標賽

### 等級提升
- 累積 Realm Points 提升 Moca ID 等級
- 解鎖更高級別的權益和獎勵
- 獲得獨家 NFT 空投

### 獨家體驗
- 優先參與 Beta 測試
- VIP 活動邀請
- 合作夥伴特殊福利

---

## 技術特點

- **鏈上記錄**：所有積分變動透明可查
- **跨鏈累積**：支援多條區塊鏈的積分統計
- **不可轉讓**：Realm Points 綁定個人身份，無法交易
