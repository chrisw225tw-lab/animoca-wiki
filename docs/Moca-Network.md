# Moca Network

**Moca Network** 是 Animoca Brands 的旗艦項目，目標是建立**全球最大的跨鏈去中心化數字身份網絡**。

| 項目 | 詳情 |
|------|------|
| **定位** | 跨鏈數字身份基礎設施 |
| **核心技術** | [[AIR-Kit]]（Animoca Identity & Reputation Kit） |
| **代幣** | $MOCA |
| **可觸達用戶** | 7億+ |
| **生態夥伴** | 570+ Animoca 投資組合公司 |

---

## 核心產品

### 1. [[Moca-ID]] — 數字身份
用戶的 Web3 通用身份，跨鏈管理資產和聲譽。

### 2. [[Moca-Proof]] — 驗證與獎勵平台
基於零知識證明的身份驗證系統，用戶可驗證成就並獲得獎勵。
- **狀態：** Beta 測試網運行中
- **主網上線：** 2026年

### 3. [[Moca-Portfolio]] — 投資組合配額平台
為 $MOCA 質押者提供 Animoca 投資組合項目的代幣配額。
- **總值：** $2,000萬+ 代幣配額
- **例子：** Magic Eden 的 $ME 代幣配額

### 4. [[Moca-Chain]] — 專用區塊鏈
為身份和用戶數據設計的 L1 區塊鏈。

## 技術架構：AIR Kit

Moca Network 建立在 [[AIR-Kit]] 之上：

| 組件 | 功能 |
|------|------|

| **I** — Identity | 去中心化身份驗證 |
| **R** — Reputation | 鏈上聲譽累積 |
