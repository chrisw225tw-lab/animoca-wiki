# OpenSea

**OpenSea** 是全球最大的 NFT 市場，也是 Web3 數字資產交易的核心基礎設施。

| 項目 | 詳情 |
|------|------|
| **成立時間** | 2017年 |
| **創辦人** | Devin Finzer、Alex Atallah |
| **總部** | 美國紐約 |
| **總交易量** | 超過 200 億美元 |
| **Animoca 角色** | 早期投資者和戰略夥伴 |

---

## 平台概述

### 市場地位
OpenSea 是 NFT 行業的先驅和領導者，支持：
- **多鏈支援**：Ethereum、Polygon、Solana、Arbitrum 等
- **多類型資產**：藝術品、收藏品、遊戲道具、虛擬土地
- **創作者工具**：無程式碼鑄造、版稅設定、收藏管理

### 核心功能
| 功能 | 說明 |
|------|------|
| 買賣交易 | 固定價格、拍賣、出價機制 |
| 聚合器 | 整合多個市場的流動性 |
| 批量操作 | 批量購買和出售 |
| 分析工具 | 價格走勢、稀有度排名 |

---

## 交易量里程碑

### 歷史數據
- **2021 年高峰**：單月交易量超過 30 億美元
- **累計交易量**：超過 200 億美元
- **用戶基數**：數百萬註冊用戶和錢包地址

### 知名交易
- Beeple 的數字藝術品
- CryptoPunks 和 Bored Ape Yacht Club
- 虛擬土地和元宇宙資產

---

## Yuga Labs 合作

### 戰略關係
OpenSea 與 [[Yuga-Labs]]（Bored Ape Yacht Club 母公司）建立了深度合作：
- **首發平台**：Yuga Labs 新系列的首選發行平台
- **版稅支持**：共同維護創作者版稅機制
- **市場推廣**：聯合營銷和社群活動

### 影響力
Yuga Labs 的 NFT 系列在 OpenSea 上佔據重要交易量，包括：
- Bored Ape Yacht Club (BAYC)
- Mutant Ape Yacht Club (MAYC)
- Otherdeed（Otherside 元宇宙土地）

## 技術創新

### Seaport 協議
- OpenSea 開源的 NFT 交易協議
- 降低 gas 費用，提升交易效率
- 支持更複雜的交易模式（如以物易物）

### 安全升級
- 引入惡意 URL 檢測
- 版稅強制執行工具
- 用戶資產保護機制

## Animoca Brands 投資

[[Animoca-Brands]] 是 OpenSea 的早期投資者之一。這項投資：
- 支持了 NFT 基礎設施的建設
- 為 Animoca 的 NFT 項目（如 [[Moca-NFT]]）提供流通渠道
- 強化了 Animoca 在數字資產領域的生態佈局
