# Magic Eden

**Magic Eden** 是 Solana 生態系統中領先的 NFT 市場，現已擴展至多鏈支援，成為全球前三大 NFT 交易平台之一。

| 項目 | 詳情 |
|------|------|
| **成立時間** | 2021年 |
| **創辦人** | Jack Lu、Zhuoxun Yin 等 |
| **總部** | 美國舊金山 |
| **主要區塊鏈** | Solana、Ethereum、Polygon、Bitcoin |
| **Animoca 角色** | 投資者和戰略夥伴 |

---

## 平台發展

### Solana 起源
Magic Eden 於 2021 年在 Solana 生態崛起：
- **低手續費優勢**：Solana 的低交易成本吸引大量用戶
- **快速交易**：秒級確認時間
- **社群驅動**：與 Solana NFT 項目緊密合作

### 多鏈擴展
2022-2023 年，Magic Eden 擴展至多鏈支援：
| 區塊鏈 | 上線時間 | 特色 |
|--------|----------|------|
| Solana | 2021年 | 主要市場， Ordinal 支援 |
| Ethereum | 2022年 | 與 OpenSea 競爭 |
| Polygon | 2023年 | 遊戲和企業 NFT |
| Bitcoin | 2023年 | Ordinals 市場 |

---

## 核心功能

### 交易功能
- **Launchpad**：新項目首發平台
- **二級市場**：買賣現有 NFT
- **拍賣系統**：英式拍賣和荷蘭拍賣
- **出價系統**：對整個收藏出價

### 創作者工具
| 工具 | 說明 |
|------|------|
| Launchpad | 無程式碼 NFT 發行 |
| 版稅強制 | 保護創作者收益 |
| 白名單管理 | 輕鬆管理允許名單 |
| 空投工具 | 向持有者分發代幣 |

---

## Ordinals 市場

### Bitcoin NFT 革命
Magic Eden 是 Bitcoin Ordinals 的主要交易平台：
- **首發優勢**：最早支援 Ordinals 的主流市場
- **交易量**：佔據 Ordinals 市場主要份額
- **創新功能**：支持 BRC-20 代幣交易

### 多協議支援
- **Ordinals**：Bitcoin 原生 NFT
- **BRC-20**：Bitcoin 同質化代幣標準
- **Runes**：新的 Bitcoin 代幣協議

---

## 遊戲生態

### 遊戲 NFT 專注
Magic Eden 在遊戲 NFT 領域表現突出：
- **專屬遊戲區**：遊戲資產分類和發現
- **跨遊戲整合**：支援多遊戲資產交易
- **遊戲夥伴**：與知名遊戲工作室合作

### 熱門遊戲項目
- **Aurory**：Solana RPG 遊戲
- **Star Atlas**：太空元宇宙
- **DeGods**：熱門 PFP 系列

## 代幣與獎勵

### Diamonds 獎勵計劃
Magic Eden 推出 Diamonds 忠誠度計劃：
- **交易獎勵**：根據交易量賺取 Diamonds
- **質押獎勵**：長期持有者額外獎勵
- **未來代幣**：Diamonds 可兌換未來治理代幣

### 代幣傳聞
市場預期 Magic Eden 可能發行平台代幣：
- 類似於 $BLUR 和 $LOOKS 的模式
- Diamonds 計劃可能是空投資格依據
