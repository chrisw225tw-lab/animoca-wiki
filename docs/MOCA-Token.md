# MOCA Token

**MOCA Token** 是 [[Moca-Network]] 的原生實用代幣，用於生態治理、質押獎勵和平台功能訪問。

| 項目 | 詳情 |
|------|------|
| **代幣名稱** | MOCA |
| **網絡** | [[Moca-Network]] |
| **類型** | 實用和治理代幣 |
| **主要用途** | 質押、治理、MocaPortfolio 訪問 |
| **發行方** | [[Animoca-Brands]] |

---

## 核心功能

### 1. 質押 (Staking)
MOCA 持有者可以質押代幣獲得多種獎勵：
- **質押獎勵**：根據質押數量和時間獲得收益
- **流動性質押**：在保持流動性的同時賺取獎勵
- **複利增長**：獎勵可自動複投

### 2. 治理權利
- **提案投票**：參與 Moca Network 的重要決策
- **參數調整**：投票決定協議參數（如手續費、獎勵率）
- **資金分配**：決定生態基金的用途

### 3. MocaPortfolio 訪問
| 權益等級 | MOCA 持有量 | 專屬權益 |
|----------|-------------|----------|
| 基礎 | 任何數量 | 基本訪問權 |
| 高級 | 門檻數量 | 優先投資額度 |
| VIP | 大量持有 | 獨家項目和服務 |

---

## 代幣經濟學

### 總供應與分配
| 分配類別 | 比例 | 說明 |
|----------|------|------|
| 生態激勵 | 35% | 質押獎勵、用戶激勵 |
| 團隊和顧問 | 20% | 長期鎖倉 |
| 投資者 | 20% | 私募和公募 |
| 金庫儲備 | 15% | 戰略發展 |
| 流動性 | 10% | 交易所流動性 |

### 解鎖時間表
- **團隊代幣**：4 年線性解鎖
- **投資者代幣**：分期解鎖，防砸盤
- **生態激勵**：持續釋放，支持長期增長

## 質押機制詳解

### 質押選項
| 類型 | 鎖定期 | 年化收益 |
|------|--------|----------|
| 靈活質押 | 無 | 較低 |
| 30 天鎖定 | 30 天 | 中等 |
| 90 天鎖定 | 90 天 | 較高 |
| 180 天鎖定 | 180 天 | 最高 |

### 質押獎勵來源
- 協議手續費的一部分
- 新發行的 MOCA 代幣
- 合作項目的代幣獎勵
