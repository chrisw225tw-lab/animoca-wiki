# Moca NFT

**Mocaverse NFT** 是 [[Animoca-Brands]] 推出的旗艦 NFT 系列，包含 8,888 個獨特的頭像 (PFP)，代表 Moca 生態的會員資格。

| 項目 | 詳情 |
|------|------|
| **總量** | 8,888 個 |
| **類型** | PFP（頭像）NFT |
| **區塊鏈** | Ethereum |
| **發行時間** | 2023年 |
| **權益** | 生態獨家福利、空投、治理 |

---

## 系列概述

### 設計特色
Mocaverse NFT 採用獨特的宇宙主題設計，每個角色都有：
- **獨特外觀**：不同的膚色、服裝、配飾組合
- **稀有度分級**：普通、稀有、史詩、傳說等級別
- **特殊特徵**：部分角色帶有聯名或限定標誌

### 發行方式
- **Allowlist 鑄造**：合作夥伴和社群成員優先
- **公開發售**：限量公開鑄造
- **二級市場**：可在 OpenSea、Blur 等平台交易
