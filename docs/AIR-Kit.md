# AIR Kit

**AIR Kit**（Animoca Identity & Reputation Kit）是 [[Moca-Network]] 的模組化數字身份基礎設施。

| 項目 | 詳情 |
|------|------|
| **全名** | Animoca Identity & Reputation Kit |
| **類型** | 模組化身份基礎設施 |
| **所屬** | [[Moca-Network]] |
| **功能** | 跨鏈管理數字身份、資產、聲譽 |

---

## AIR 三大組件

```
┌─────────────────────────────────────────┐
│              AIR Kit                     │
├─────────────┬─────────────┬─────────────┤
│    A        │      I      │      R      │
│  Account    │  Identity   │  Reputation │
│  賬戶管理    │   身份驗證   │   聲譽累積   │
└─────────────┴─────────────┴─────────────┘
```

### A — Account（賬戶）
- **跨鏈賬戶抽象（Account Abstraction）**
- 一個身份管理多鏈資產
- 簡化用戶體驗，無需為每條鏈創建單獨賬戶

### I — Identity（身份）
- **去中心化身份驗證**
- 使用零知識證明（Zero-Knowledge Proofs）
- 驗證身份屬性（如「我是真人」）而不暴露個人數據
- 符合合規標準，保護數據所有權

### R — Reputation（聲譽）
- **鏈上聲譽系統**
- 根據行為累積 Realm Points
- 可驗證的成就系統
- 企業可基於聲譽分發獎勵、篩選社群

## 技術特點

| 特點 | 說明 |
|------|------|
| **跨鏈互操作** | 支援多條區塊鏈 |
| **隱私保護** | 零知識證明技術 |
| **模組化設計** | 企業可靈活整合 |
| **可組合性** | 與其他 Web3 應用無縫整合 |

---

## 應用場景

### 對用戶
- 統一的 Web3 身份
- 累積跨平台聲譽
- 保護隱私的同時驗證身份

### 對企業
- 基於聲譽的用戶篩選
- 精準分發獎勵
- 合規的身份驗證
