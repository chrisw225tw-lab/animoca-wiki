# SAND Token

**SAND** 是 [[The-Sandbox]] 元宇宙平台的原生實用代幣，用於土地購買、資產交易和生態治理。

| 項目 | 詳情 |
|------|------|
| **代幣名稱** | SAND |
| **平台** | [[The-Sandbox]] |
| **類型** | ERC-20 實用代幣 |
| **總供應量** | 30 億枚 |
| **區塊鏈** | Ethereum / [[Polygon]] |

---

## 核心功能

### 1. LAND 購買
SAND 是購買 The Sandbox 虛擬土地的主要貨幣：
- **土地拍賣**：新 LAND 發售時使用 SAND 競標
- **二級市場**：在 OpenSea 等平台用 SAND 購買
- **地產開發**：購買多塊土地合併為 Estate

### 2. 資產交易
| 用途 | 說明 |
|------|------|
| NFT 購買 | 購買遊戲內資產和裝備 |
| 創作者收入 | 出售原創內容賺取 SAND |
| 服務支付 | 聘請建築師和設計師 |

### 3. 質押獎勵
SAND 持有者可以通過質押獲得：
- **被動收入**：根據質押數量賺取收益
- **GEM 代幣**：質押獎勵的一部分
- **CATALYST 代幣**：用於創建資產的特殊代幣

## 使用場景詳解

### 創作者經濟
- **資產鑄造**：使用 SAND 在 VoxEdit 創建 NFT
- **遊戲變現**：設計遊戲收取 SAND 入場費
- **土地租賃**：出租 LAND 賺取 SAND 租金

### 玩家體驗
- **遊戲入場**：部分遊戲需要 SAND 入場費
- **道具購買**：購買裝備和消耗品
- **活動參與**：特殊活動的門票支付

### 治理參與
- **DAO 投票**：參與平台治理決策
- **基金會撥款**：投票決定資金用途
- **規則制定**：影響平台發展方向

## 質押計劃

### SAND 質押
| 方案 | 特點 | 獎勵 |
|------|------|------|
| 基礎質押 | 無鎖定期 | SAND 獎勵 |
| 高級質押 | 固定鎖定期 | SAND + GEM |
| LAND 質押 | 質押 SAND 在 LAND 上 | 額外獎勵加成 |

### 獎勵代幣
- **GEM**：用於提升資產屬性
- **CATALYST**：決定資產稀有度
- **mSAND**：Polygon 上的封裝版本

## 市場表現

### 歷史亮點
- 2021 年元宇宙熱潮中創下歷史高價
- 市值一度進入前 50 大加密貨幣
- 廣泛上市於主要交易所

### 生態價值
- SAND 的價值與 The Sandbox 生態發展掛鉤
- LAND 銷售和活動帶動代幣需求
- 品牌合作提升代幣認知度
