# Stablecoin（穩定幣）

**Stablecoin**（穩定幣）是與法定貨幣（通常是美元）掛鉤的加密貨幣，結合了傳統貨幣的穩定性和區塊鏈的便利性。

---

## 穩定幣類型

### 1. 法幣抵押型
由實際法幣儲備支持：
| 穩定幣 | 發行方 | 儲備 |
|--------|--------|------|
| USDT | Tether | 美元及等價物 |
| USDC | Circle | 現金和美國國債 |
| FDUSD | First Digital | 現金儲備 |

**特點**：最流行，流動性最好，但中心化風險

### 2. 加密資產抵押型
由加密貨幣超額抵押：
| 穩定幣 | 平台 | 機制 |
|--------|------|------|
| DAI | MakerDAO | ETH 等超額抵押 |
| sUSD | Synthetix | SNX 抵押 |

**特點**：去中心化，但需要超額抵押

### 3. 算法型
通過算法調節供應量：
- **成功案例**：少數項目成功維持掛鉤
- **失敗案例**：UST（2022 年崩盤）

**特點**：創新但風險較高

## 主要穩定幣

### USDT (Tether)
| 項目 | 詳情 |
|------|------|
| **發行方** | Tether Limited |
| **上線時間** | 2014年 |
| **市值** | 約 $800-900 億（最大）|
| **區塊鏈** | 多鏈發行 |

**特點**：
- 市場份額最大
- 流動性最好
- 儲備透明度曾受質疑

### USDC (USD Coin)
| 項目 | 詳情 |
|------|------|
| **發行方** | Circle（與 Coinbase 合作）|
| **上線時間** | 2018年 |
| **市值** | 約 $300-400 億 |
| **合規性** | 最合規的穩定幣之一 |

**特點**：
- 監管合規程度高
- 儲備透明，定期審計
- 機構首選

## 使用場景

### 交易和投資
| 用途 | 說明 |
|------|------|
| 避險工具 | 市場波動時保值 |
| 交易對 | 加密貨幣交易的基準貨幣 |
| 套利 | 跨交易所價差套利 |
| 入金 | 進入加密市場的入口 |

### 支付和匯款
- **跨境支付**：24/7 即時到賬，費用低
- **工資支付**：遠程工作者收款
- **電商**：加密原生支付
- **匯款**：替代傳統匯款服務

### 去中心化金融 (DeFi)
- 借貸協議的抵押品和借款資產
- 流動性挖礦的穩定收益來源
- 收益農場的基礎資產

## 穩定幣的優勢

| 優勢 | 說明 |
|------|------|
| 價格穩定 | 避免加密貨幣的劇烈波動 |
| 區塊鏈速度 | 即時轉帳，無銀行營業時間限制 |
| 全球可及 | 只需網絡連接，無需銀行帳戶 |
| 透明可審計 | 鏈上記錄公開可查 |
| 可編程 | 可整合到智能合約 |

---

## 風險與挑戰

### 中心化風險
- 發行方可能破產或凍結資金
- 監管打擊可能影響運營
- 儲備資產的質量問題

### 監管不確定
- 各國監管政策不一致
- 可能面臨證券分類
- 反洗錢合規要求

### 脫鉤風險
- 極端市場條件下可能脫離掛鉤
- 儲備不足導致擠兌
- 信心危機引發連鎖反應
