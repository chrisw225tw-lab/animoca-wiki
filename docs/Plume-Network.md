# Plume Network

**Plume Network** 是專注於真實世界資產（RWA）代幣化的區塊鏈網絡，與 Animoca Brands 合作為機構和零售用戶提供 RWA 收益產品。

| 項目 | 詳情 |
|------|------|
| **類型** | RWA 專用 Layer 1 |
| **定位** | 機構級 RWA 收益平台 |
| **合作方** | [[Animoca-Brands]] |
| **用戶觸達** | 7億+（通過 Animoca 生態）|

---

## 網絡簡介

Plume Network 是第一個專為 RWA 設計的模塊化區塊鏈，致力於簡化真實世界資產的代幣化流程，連接傳統金融與 DeFi 生態。

---

## 核心特點

### 1. RWA 專屬設計
- 原生支持資產代幣化
- 合規框架內建
- 機構級安全標準

### 2. 收益優化
- 與傳統金融產品連接
- 鏈上鏈下收益整合
- 自動化收益分配

### 3. 生態整合
- 與現有 DeFi 協議兼容
- 跨鏈資產轉移
- 流動性聚合

---

## 與 Animoca Brands 的合作

### 合作內容
- **用戶導流** — Animoca 7億+用戶可使用 Plume 服務
- **機構收益** — 共同開發機構級 RWA 產品
- **生態整合** — [[Moca-Network]] 與 Plume 的深度合作

### 合作意義
此合作將 Plume 的 RWA 基礎設施與 Animoca 龐大的用戶基礎結合，為零售用戶開啟機構級收益產品的入口。
