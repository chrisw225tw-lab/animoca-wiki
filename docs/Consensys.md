# Consensys

**Consensys** 是以太坊生態系統中最重要的軟件公司之一，開發了 MetaMask、Infura 等核心 Web3 基礎設施。

| 項目 | 詳情 |
|------|------|
| **類型** | Web3 軟件公司 |
| **成立時間** | 2014年 |
| **創辦人** | Joseph Lubin（以太坊聯合創始人）|
| **投資方** | [[Animoca-Brands]] |

---

## 公司簡介

Consensys 由以太坊聯合創始人 Joseph Lubin 創立，是推動以太坊生態發展的關鍵力量。公司開發了多款被廣泛使用的 Web3 工具，為開發者和用戶提供進入去中心化世界的入口。

---

## 核心產品

### 1. MetaMask
- 全球最受歡迎的加密貨幣錢包
- 瀏覽器擴展和移動應用
- 3000萬+ 月活用戶
- 大多數 Web3 應用的默認入口

### 2. Infura
- 以太坊節點即服務
- 為開發者提供可靠的區塊鏈訪問
- 支持多條 EVM 兼容鏈
- 服務數萬個 Web3 項目

### 3. Linea
- Consensys 開發的 zkEVM Layer 2
- 以太坊擴展解決方案
- 與 MetaMask 深度整合

## 行業影響

Consensys 的產品是 Web3 基礎設施的基石：
- MetaMask 是絕大多數用戶的首個 Web3 錢包
- Infura 支撐了以太坊生態的主要流量
- 開發者工具降低了 DApp 開發門檻
