# Robby Yung

**Robby Yung** 是 [[Animoca-Brands]] 的北美營運行政總裁（CEO of North American Operations），同時擔任 Animoca Brands KK（日本分公司）的行政總裁，負責公司在北美和日本市場的業務發展。

## 職業經歷

### Animoca Brands（2014年至今）

Robby Yung 於2014年加入 [[Animoca-Brands]]，自公司成立初期便擔任重要領導角色。

**2014-2021年：Animoca Brands 行政總裁**
- 帶領公司從傳統手機遊戲開發商轉型為區塊鏈遊戲和 NFT 領域的全球領導者
- 主導多項重大投資和收購項目
- 推動公司估值達到獨角獸級別（超過10億美元）

**2021年至今：北美營運行政總裁**
- 專注於美國和加拿大市場的業務擴展
- 管理公司在北美的子公司，包括 [[The-Sandbox]]、Blowfish Studios 等
- 負責日本市場的戰略發展，領導 Animoca Brands KK

## 負責業務

- 監督美國和加拿大的營運和投資活動

- 領導 Animoca Brands KK（日本分公司）
