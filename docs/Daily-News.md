### 🏢 Animoca Brands 重大動態

#### 產品發布：Animoca Minds（2月5日）

Animoca Brands 正式宣布與 **CryptoSlam 的 Ethoswarm** 建立戰略合作，推出 **Animoca Minds**，讓用戶部署和運營持久 AI 代理的新服務。

| 功能 | 描述 |
|------|------|
| **發布日期** | 2026年2月5日（搶先體驗） |
| **合作夥伴** | Ethoswarm / CryptoSlam |
| **核心功能** | 無需技術設置即可部署持久、自主的 AI 代理 |
| **用戶界面** | 電郵和 Telegram 整合（自然語言） |
| **目標用戶** | 非技術消費者 |

**主要差異化特點：**
- **無需硬件：** 完全在雲端運行
- **專業代理配置：** 市場研究員、加密偵察員等預設代理
- **簡化入門：** 通過白標體驗「喚醒」個性化代理

**高管評論：**
> *「我們正在超越真正的 AI 代理能力僅限於工程師和私人服務器的階段。我們的目標是通過利用強大的基礎框架，讓任何人都能無縫啟動持久的自主代理，從而實現創新的民主化。」* — Mohamed Ezeldin，Animoca Labs 負責人

> *「AI 代理扭轉了這種關係：代理通過學習您的上下文並幫助您執行操作來適應您，同時放大您的判斷力。」* — 蕭逸（Yat Siu），聯合創辦人兼執行主席
