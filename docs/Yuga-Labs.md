# Yuga Labs

**Yuga Labs** 是 Web3 領域最具影響力的公司之一，旗下擁有多個頂級 NFT 項目，包括 Bored Ape Yacht Club 和 CryptoPunks。

| 項目 | 詳情 |
|------|------|
| **類型** | NFT 與元宇宙公司 |
| **知名項目** | BAYC、CryptoPunks、Meebits |
| **投資方** | [[Animoca-Brands]] |
| **行業地位** | NFT 領域領導者 |

---

## 公司簡介

Yuga Labs 成立於 2021 年，迅速成為 NFT 行業的標誌性企業。公司通過收購和原創，建立了一個強大的 IP 組合，並積極拓展元宇宙和遊戲領域。

---

## 核心資產

### 1. Bored Ape Yacht Club (BAYC)
- 10,000 個獨特的猿猴 NFT
- 明星和名人持有者眾多
- 持有者享有專屬社群和活動權益

### 2. CryptoPunks
- 2017 年推出的 OG NFT 項目
- 10,000 個像素風格頭像
- NFT 文化的奠基者之一

### 3. Meebits
- 3D 體素角色 NFT
- 適合元宇宙和遊戲使用
- 20,000 個獨特角色

## 社群與文化

Yuga Labs 的成功很大程度上歸功於其強大的社群文化：
- 持有者專屬 Discord 伺服器
- 線下聚會和活動（ApeFest）
- 商業授權（持有者可用於商業用途）

---

## 與 Animoca Brands 的關係

[[Animoca-Brands]] 是 Yuga Labs 的早期投資者和重要合作夥伴：
- 參與 Yuga Labs 的融資輪次
- 在元宇宙和遊戲領域探索合作
- 共同推動 NFT 和 Web3 的大規模採用
