# Open Campus

**Open Campus** 是去中心化教育平台，致力於革新傳統教育模式，讓學習者掌握自己的教育數據和證書。

| 項目 | 詳情 |
|------|------|
| **定位** | 去中心化教育平台 |
| **開發商** | [[Animoca-Brands]] |
| **核心概念** | 教育數據主權 |
| **代幣** | EDU |

---

## 平台簡介

Open Campus 利用區塊鏈技術創建一個開放、透明且由社群驅動的教育生態系統。平台讓學習者擁有並控制自己的教育記錄，同時為教育工作者提供新的變現途徑。

---

## 核心功能

### 1. 去中心化教育記錄
- 學歷證書上鏈存證
- 防偽可驗證的學習證明
- 終身學習檔案管理

### 2. 教育者變現
- 課程內容代幣化
- 直接與學習者連接
- 去除中間商抽成

### 3. 社群治理
- EDU 代幣持有者參與決策
- 教育內容策展機制
- 獎學金和資助計劃

---

## 生態系統組成

| 角色 | 功能 |
|------|------|
| **學習者** | 獲取知識、累積可驗證憑證 |
| **教育者** | 創建和銷售課程內容 |
| **合作機構** | 認可和採納平台憑證 |
| **內容創作者** | 貢獻教育資源 |

---

## 技術特點

- **智能合約** — 自動化課程購買和收益分配
- **NFT 證書** — 獨一無二的學習成就證明
- **數據主權** — 用戶完全控制自己的教育數據
- **跨平台整合** — 與現有教育工具兼容

Open Campus 已與多家教育機構和企業建立合作關係，推動區塊鏈技術在教育領域的實際應用。
