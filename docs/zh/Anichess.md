# Anichess

**Anichess** 是 Animoca Brands 開發的區塊鏈國際象棋遊戲，將傳統棋藝與 Web3 技術創新結合。

| 項目 | 詳情 |
|------|------|
| **類型** | 區塊鏈策略遊戲 |
| **開發商** | [[Animoca-Brands]] |
| **遊戲類型** | 國際象棋 |
| **特點** | 結合 NFT 與邊玩邊賺 |

---

## 遊戲簡介

Anichess 在經典國際象棋的基礎上引入區塊鏈元素，玩家可以擁有獨特的棋子 NFT，參與競技對戰並獲得獎勵。遊戲既保留了傳統象棋的策略深度，又增添了數字產權和經濟激勵。

---

## 核心特色

### 1. 棋子 NFT
- 每個棋子都是獨一無二的 NFT
- 不同稀有度和屬性
- 可交易、收藏和升級

### 2. 邊玩邊賺（Play-to-Earn）
- 參與對戰贏取獎勵
- 錦標賽獎金池
- 質押棋子獲得收益

### 3. 創新玩法
- 特殊技能和魔法道具
- 多樣化的遊戲模式
- PvP 競技與 PvE 挑戰

---
