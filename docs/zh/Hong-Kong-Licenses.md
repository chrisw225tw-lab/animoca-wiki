# Animoca Brands 香港牌照

本頁記錄 [[Animoca-Brands]] 在香港的監管狀態和牌照申請進度。

## 公司背景

### 總部位置
[[Animoca-Brands]] 總部位於香港，是亞洲最大的 Web3 和數字資產公司之一。

| 項目 | 詳情 |
|------|------|
| **總部** | 香港 |
| **監管環境** | 香港證監會（SFC）框架下 |
| **戰略地位** | 連接東西方市場的樞紐 |

## 現有監管狀態

### 證監會牌照
Animoca Brands 持有多項香港證監會牌照：

| 牌照類型 | 允許業務 | 狀態 |
|----------|----------|------|
| **1 號牌** | 證券交易 | 持有 |
| **4 號牌** | 就證券提供意見 | 持有 |
| **9 號牌** | 資產管理 | 持有 |
| **VASP 牌照** | 虛擬資產服務 | 申請中/持有 |

### 合規承諾
Animoca Brands 始終將合規置於優先位置：
- 與監管機構保持積極溝通
- 投資組合項目遵循當地法規
- 透明的業務運營

## 穩定幣牌照進度

### 2025-2026 計劃
作為四大戰略方向之一，穩定幣發行需要相應牌照：

| 階段 | 時間 | 里程碑 |
|------|------|--------|
| **準備階段** | 2024-2025 | 與監管機構初步溝通 |
| **申請階段** | 2025 | 正式提交牌照申請 |
| **合作階段** | 2025-2026 | 與渣打、HKT 建立合作 |
| **發行準備** | 2026 | 技術和儲備準備就緒 |

### 合作夥伴的監管優勢
| 合作夥伴 | 監管角色 |
|----------|----------|
| [[Standard-Chartered]] | 銀行合規、儲備託管 |
| [[HKT]] | 本地分發、KYC/AML |

這種合作模式有助於滿足香港金管局對穩定幣發行人的嚴格要求。

---

## 香港穩定幣監管框架

### 法規背景
香港金管局（HKMA）於 2024 年建立穩定幣發行人牌照制度：

| 要求 | 說明 |
|------|------|
| **儲備管理** | 全額儲備，高流動性資產 |
| **贖回權利** | 持有人可隨時按面值贖回 |
| **審計要求** | 定期審計和披露 |
| **資本要求** | 最低資本金要求 |

### 發行人類別
| 類別 | 要求 |
|------|------|
| **本地發行人** | 在香港註冊，獲得牌照 |
| **海外發行人** | 在認可司法管轄區註冊 |
| **銀行發行人** | 受監管銀行可直接發行 |

## RWA 代幣化合規

### 監管考量
Animoca 的 RWA 戰略同樣需要關注香港法規：

| 資產類型 | 監管機構 | 要求 |
|----------|----------|------|
| 證券類代幣 | SFC | 證券法合規 |
| 基金代幣 | SFC | 基金牌照 |
| 不動產代幣 | 多機構 | 物業和證券法 |

### 合規策略
- 與持牌機構合作（如 [[Rayls]]）
- 僅服務專業投資者（如適用）
- 透明的資產託管和審計

## 香港的戰略優勢

### 為何選擇香港
| 優勢 | 說明 |
|------|------|
| **監管清晰** | SFC 和 HKMA 提供明確指引 |
| **國際樞紐** | 連接中國內地與全球市場 |
| **人才密集** | 金融和技術人才匯聚 |
| **資金通道** | 國際資本進出便利 |

### 與其他地區比較
| 地區 | 優勢 | 挑戰 |
|------|------|------|
| 香港 | 監管清晰、亞洲樞紐 | 市場規模有限 |
| 新加坡 | 機構友好 | 零售限制嚴格 |
| 美國 | 市場大 | 監管不確定 |
| 歐盟 | MiCA 框架 | 合規成本高 |
