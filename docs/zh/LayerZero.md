# LayerZero

**LayerZero** 是領先的跨鏈互操作性協議，旨在實現不同區塊鏈之間的無縫通信和資產轉移。

| 項目 | 詳情 |
|------|------|
| **類型** | 跨鏈互操作性協議 |
| **核心技術** | 超輕節點（Ultra Light Node）|
| **投資方** | [[Animoca-Brands]] |
| **支援鏈數** | 50+ 條區塊鏈 |

---

## 協議簡介

LayerZero 提供了一種全新的跨鏈通信方式，通過創新的「超輕節點」設計，在保證安全性的同時實現低成本、高效率的跨鏈消息傳遞。

---

## 核心技術

### 超輕節點（Ultra Light Node）
- 無需在每條鏈運行完整節點
- 通過預言機和中繼器驗證交易
- 極大降低跨鏈成本

### 雙重驗證機制
| 組件 | 功能 |
|------|------|
| **Oracle** | 提供區塊頭信息 |
| **Relayer** | 提供交易證明 |
| **安全機制** | 兩者獨立，需同時驗證 |

---

## 主要應用

### 1. 跨鏈橋（Stargate）
- 基於 LayerZero 的跨鏈 DEX
- 原生資產跨鏈轉移
- 統一流動性池

### 2. Omnichain Fungible Token（OFT）
- 可在多鏈流通的代幣標準
- 無需包裝（wrapping）
- 保持代幣同質性

### 3. Omnichain NFT（ONFT）
- 可在多鏈間轉移的 NFT
- 保持元數據一致性
- 無縫的跨鏈體驗
