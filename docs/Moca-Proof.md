# Moca Proof

**Moca Proof** 是 Animoca Brands 開發的數字身份驗證與獎勵平台，為用戶提供可驗證的鏈上身份憑證。

| 項目 | 詳情 |
|------|------|
| **定位** | 數字身份驗證與獎勵平台 |
| **測試網狀態** | Beta 版本已上線 |
| **主網預計** | 2026年 |
| **所屬生態** | [[Moca-Network]] |

---

## 核心功能

### 1. 身份驗證
- 提供可驗證的數字身份憑證
- 基於零知識證明技術保護隱私
- 支持多鏈身份統一驗證

### 2. 獎勵機制
- 用戶通過完成驗證獲得獎勵
- 與 [[Moca-ID]] 生態積分系統整合
- 激勵合規的身份驗證行為

### 3. 跨鏈互操作
- 驗證結果可在多條區塊鏈上使用
- 與 [[AIR-Kit]] 技術深度整合
- 支持主流公鏈生態

---

## 技術架構

Moca Proof 建立在 [[Moca-Chain]] 之上，利用 Layer 1 區塊鏈的安全性和效率，為身份驗證提供堅實的基礎設施支持。

---

## 應用場景

- **KYC 驗證** — 簡化金融服務的身份驗證流程
- **年齡驗證** — 遊戲和內容平台的合規驗證
- **資格證明** — 活動參與和空投資格確認
- **聲譽系統** — 鏈上聲譽評分與驗證
