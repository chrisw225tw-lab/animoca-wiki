# John Madden

**John Madden** 是 [[Animoca-Brands]] 的公司秘書（Company Secretary）兼財務顧問，曾於2020年擔任代理財務總監（acting CFO）。他在公司治理和財務管理方面擁有豐富經驗，並於2025年12月被任命為公司董事會非執行董事。

| 項目 | 詳情 |
|------|------|
| **現任職位** | 公司秘書（Company Secretary）、非執行董事 |
| **曾任職位** | 代理財務總監（acting CFO，2020年） |
| **公司** | [[Animoca-Brands]] |
| **任命董事** | 2025年12月10日 |
| **專長** | 公司治理、財務管理、合規 |

## 職業經歷

### Animoca Brands（2019年至今）

#### 財務顧問與公司秘書（2019年至今）
John Madden 於2019年加入 [[Animoca-Brands]]，擔任財務顧問和聯席公司秘書：
- 協助管理公司財務運作
- 負責公司治理和合規事務
- 支持董事會運作和決策流程

#### 代理財務總監（2020年）
2020年3月，John Madden 被任命為代理財務總監（acting CFO）：
- 領導財務團隊日常運作
- 管理財務報告和預算
- 支持投資者關係活動

此後，[[Jared-Shaw]] 於2022年9月加入擔任正式財務總監。

#### 董事會任命（2025年12月）
2025年12月10日，John Madden 被任命為 Animoca Brands 董事會非執行董事：
- 為董事會提供財務和公司治理專業知識
- 參與公司戰略決策
- 協助監督風險管理和合規事務
