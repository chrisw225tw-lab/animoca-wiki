# Standard Chartered

**Standard Chartered（渣打銀行）** 是 Animoca Brands 在穩定幣業務上的重要銀行合作夥伴，共同推動合規數字貨幣的發展。

| 項目 | 詳情 |
|------|------|
| **類型** | 國際銀行集團 |
| **總部** | 英國倫敦 |
| **合作領域** | 穩定幣發行與託管 |
| **合作方** | [[Animoca-Brands]]、[[HKT]] |

---

## 銀行簡介

渣打銀行成立於 1853 年，是一家專注於亞洲、非洲和中東市場的國際銀行集團。在數字資產領域，渣打銀行積極佈局，提供加密貨幣託管、交易和結算服務。

---

## 與 Animoca Brands 的合作

### 穩定幣合作項目
- **角色** — 合規穩定幣發行的銀行合作夥伴
- **服務** — 法幣儲備託管、合規審查
- **目標** — 推出面向機構和零售用戶的穩定幣

### 合作架構
| 合作方 | 角色 |
|--------|------|
| [[Animoca-Brands]] | 項目主導與生態整合 |
| **Standard Chartered** | 銀行服務與合規支持 |
| [[HKT]] | 支付渠道與用戶觸達 |

---

## 數字資產戰略

渣打銀行在數字資產領域的佈局包括：
- **託管服務** — 為機構客戶提供加密資產託管
- **交易服務** — 加密貨幣場外交易（OTC）
- **結算服務** — 數字資產交易結算
- **穩定幣** — 支持合規穩定幣發行

---

## 香港市場

渣打銀行在香港擁有深厚根基：
- 發鈔銀行之一
- 與香港金管局緊密合作
- 參與香港數字資產監管框架建設

---

## 合作意義

此合作標誌著：
- 傳統金融機構對 Web3 的認可
- 穩定幣合規化的重要進展
- 香港作為數字資產中心的地位提升
