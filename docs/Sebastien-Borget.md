# Sebastien Borget

**Sebastien Borget** 是 [[The-Sandbox]] 的聯合創辦人兼營運總監（COO），同時擔任區塊鏈遊戲聯盟（Blockchain Game Alliance）主席，是元宇宙和區塊鏈遊戲領域的重要人物。

| **公司** | [[The-Sandbox]]（[[Animoca-Brands]] 子公司） |
| **兼任職位** | 區塊鏈遊戲聯盟（BGA）主席 |

## 職業經歷

### The Sandbox（2012年至今）

#### 創立初期（2012年-2018年）
Sebastien Borget 於2012年與 [[Arthur-Madrid]] 共同創立 Pixowl，開發手機遊戲《The Sandbox》：
- 開發像素風格的沙盒遊戲
- 累積數百萬下載量
- 建立創意遊戲社群

#### 區塊鏈轉型（2018年至今）
2018年，[[Animoca-Brands]] 收購 Pixowl，Sebastien Borget 領導 The Sandbox 向區塊鏈元宇宙轉型：
- 開發基於區塊鏈的3D元宇宙平台
- 推出 SAND 代幣和 LAND NFT
- 建立創作者經濟生態

### 區塊鏈遊戲聯盟（Blockchain Game Alliance）
**主席（President）**

Sebastien Borget 領導區塊鏈遊戲聯盟，推動：
- 區塊鏈遊戲行業標準制定
- 會員之間的合作與知識分享
- 向傳統遊戲行業推廣區塊鏈技術
- 與監管機構和政策制定者溝通

## The Sandbox 發展

### 元宇宙建設
在 Sebastien Borget 的領導下，The Sandbox 發展成為領先的元宇宙平台：

| 里程碑 | 說明 |
|--------|------|
| **LAND 銷售** | 虛擬土地 NFT 銷售創下紀錄 |
| **品牌合作** | 與華納音樂、Adidas、Snoop Dogg 等合作 |
| **創作者工具** | 推出 VoxEdit 和 Game Maker |
| **市場平台** | 建立 NFT 交易市場 |
