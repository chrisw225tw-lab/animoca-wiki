# TON Foundation

**TON Foundation** 是 The Open Network (TON) 區塊鏈生態的管理組織。

## TON 生態

TON（The Open Network）是由 Telegram 創始團隊開發的高性能 Layer 1 區塊鏈，後由社群接管並開源發展。

### 核心特點

| 特點 | 說明 |
|------|------|
| **高性能** | 支援每秒百萬級交易 |
| **分片架構** | 動態分片擴展 |
| **低手續費** | 適合微支付和頻繁交易 |
| **Telegram 整合** | 與 Telegram 生態深度結合 |

## 與 Moca Network 的合作

Moca Network 與 TON Foundation 合作，將數字身份和聲譽系統整合到 TON 生態：

- **Moca ID 登入**：TON 生態應用可使用 Moca ID 驗證
- **Realm Points**：TON 用戶可獲得和使用 Realm Points
- **跨鏈身份**：Moca ID 支援 TON 地址綁定
