# Moca ID

**Moca ID** 是你在無國界 Web3 世界的數字身份。

| 項目 | 詳情 |
|------|------|
| **定位** | 跨鏈通用數字身份 |
| **技術基礎** | [[AIR-Kit]] |
| **所屬網絡** | [[Moca-Network]] |
| **特點** | 一個身份，多鏈通行 |

---

## 核心功能

### 1. 跨鏈身份管理
- 一個 Moca ID 管理 Ethereum、Solana、Cosmos 等多條鏈的資產
- 無需在每條鏈重新建立身份

### 2. 聲譽系統（Realm Points）
- 通過參與合作夥伴的體驗賺取 Realm Points
- 鏈上行為累積聲譽

### 3. 隱私保護
- 使用零知識證明技術
- 驗證身份屬性而不暴露原始數據

---

## 如何獲得 Moca ID

1. 訪問 [moca.network](https://moca.network)
2. 連接錢包
3. 創建 Moca ID 配置文件
4. 開始賺取 Realm Points
