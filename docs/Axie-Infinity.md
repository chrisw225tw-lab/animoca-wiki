# Axie Infinity

**Axie Infinity** 是由 Sky Mavis 開發的區塊鏈遊戲，開創了「邊玩邊賺」(Play-to-Earn) 模式，成為 Web3 遊戲的現象級產品。

| 項目 | 詳情 |
|------|------|
| **開發商** | Sky Mavis |
| **上線時間** | 2018年 |
| **區塊鏈** | Ethereum（後遷移至 Ronin 側鏈） |
| **代幣** | AXS、SLP、MEC |
| **Animoca 角色** | 早期投資者和戰略夥伴 |

---

## Sky Mavis 公司

### 公司背景
Sky Mavis 是越南的區塊鏈遊戲開發商，由 Trung Nguyen 於 2018 年創立。公司使命是通過遊戲創造經濟機會，特別是為發展中國家的玩家。

### 核心團隊
- **Trung Nguyen**：CEO 兼聯合創辦人
- **Aleksander Larsen**：COO 兼聯合創辦人
- **Jeffrey Zirlin**：成長主管（社區稱為「Jihoz」）

---

## 遊戲機制

### 寵物系統
- **Axies**：遊戲中的幻想生物，以 NFT 形式存在
- **繁殖系統**：玩家可以繁殖 Axies 創造新寵物
- **戰鬥系統**：回合制策略對戰

### 邊玩邊賺模式
| 活動 | 獎勵 |
|------|------|
| 冒險模式 | SLP 代幣 |
| 競技場對戰 | SLP + AXS 代幣 |
| 繁殖 Axies | 出售新 Axies 獲利 |
| 土地遊戲 | 資源和代幣獎勵 |

---

## 代幣經濟

### AXS (Axie Infinity Shards)
- **功能**：治理代幣，用於投票和質押
- **供應量**：2.7 億枚
- **用途**：生態治理、質押獎勵、遊戲內支付

### SLP (Smooth Love Potion)
- **功能**：遊戲內貨幣，用於繁殖 Axies
- **特點**：無上限供應，通過遊戲產出
- **經濟循環**：繁殖消耗 SLP，維持代幣需求

### MEC (Mavis Eco-Connect)
- **功能**：Sky Mavis 生態系統代幣
- **用途**：跨遊戲和應用的通用貨幣
- **整合**：連接 Axie Infinity 和其他 Sky Mavis 產品

---

## 菲律賓現象

### 經濟影響
- Axie Infinity 在菲律賓爆紅，成為疫情期間的收入來源
- 許多玩家通過遊戲收入超過當地最低工資
- 催生了「Axie 公會」模式，投資者提供 NFT 給玩家使用

### 社會意義
- 展示了區塊鏈遊戲在發展中國家的潛力
- 證明了「邊玩邊賺」模式的商業可行性
- 引發了對於遊戲經濟學的廣泛討論
