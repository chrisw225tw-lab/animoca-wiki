# Cosmic Club

**Cosmic Club** 是專為 [[Moca-ID]] 持有者打造的遊戲平台，提供獨家遊戲體驗和競技錦標賽。

| 項目 | 詳情 |
|------|------|
| **定位** | Moca ID 專屬遊戲平台 |
| **主打遊戲** | Cosmic Bomber（PvP 對戰） |
| **代幣系統** | [[Realm-Points]] |
| **特色** | 獨家錦標賽、社群競技 |

---

## 核心遊戲：Cosmic Bomber

### 遊戲概述
Cosmic Bomber 是一款快節奏的 PvP 對戰遊戲，靈感源自經典炸彈人玩法，加入 Web3 元素和 NFT 角色。

### 遊戲特色
- **實時對戰**：與全球玩家 1v1 或組隊競技
- **NFT 角色**：使用 Mocaverse NFT 作為遊戲角色
- **技能升級**：通過遊戲解鎖特殊能力和道具
- **排行榜系統**：季度排名獎勵頂尖玩家

### 遊戲模式
| 模式 | 說明 |
|------|------|
| 排位賽 | 競技排名，賽季獎勵 |
| 友誼賽 | 與朋友輕鬆對戰 |
| 錦標賽 | 限時賽事，高額獎池 |
| 合作模式 | PvE 挑戰關卡 |

---

## 獨家錦標賽

### 定期賽事
- **每週挑戰賽**：小型賽事，快速對決
- **月度大賽**：大型錦標賽，豐厚獎勵
- **季度總決賽**：頂尖選手角逐總冠軍

### 獎勵機制
- **Realm Points**：參賽和獲勝均可獲得
- **獨家 NFT**：限量版遊戲道具和皮膚
- **MOCA 代幣**：高級賽事的獎池分配

---

## 社群功能

### 公會系統
- 創建或加入玩家公會
- 公會專屬賽事和獎勵
- 團隊合作完成挑戰

### 社交互動
- 遊戲內聊天和好友系統
- 戰績分享和回放
- 直播和觀戰模式

---

## 如何加入

1. 擁有 [[Moca-ID]] 數字身份
2. 訪問 Cosmic Club 官方網站
3. 連接錢包驗證身份
4. 開始遊戲賺取 [[Realm-Points]]
