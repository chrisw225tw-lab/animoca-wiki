## 在 Animoca Brands 的角色

### Moca Network 項目負責人
作為 Moca Network 的 Project Lead，Kenneth Shek 主導以下產品的開發和推廣：

- [[Moca-ID]] — 數字身份系統
- [[Moca-Proof]] — 驗證與獎勵平台
- [[Moca-Portfolio]] — 投資組合配額平台
- [[Moca-Chain]] — 專用 L1 區塊鏈
- [[AIR-Shop]] — 忠誠度平台
