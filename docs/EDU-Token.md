# EDU Token

**EDU** 是 [[Open-Campus]] 去中心化教育平台的原生代幣，推動教育內容的民主化和學習者的經濟激勵。

| 項目 | 詳情 |
|------|------|
| **代幣名稱** | EDU |
| **平台** | [[Open-Campus]] |
| **類型** | 實用和治理代幣 |
| **總供應量** | 10 億枚 |
| **區塊鏈** | Ethereum / BNB Chain |

---

## 核心願景

### 去中心化教育
Open Campus 和 EDU 代幣的目標是：
- **內容所有權**：讓教育者擁有並變現自己的內容
- **學習激勵**：獎勵認真學習的學生
- **全球可及**：打破地理和經濟障礙

---

## 代幣功能

### 1. 教育憑證
EDU 用於購買和驗證教育憑證：
| 用途 | 說明 |
|------|------|
| 課程購買 | 購買優質教育內容 |
| 證書鑄造 | 將完成證書上鏈 |
| 技能驗證 | 支付專業認證費用 |

### 2. Learn-to-Earn（邊學邊賺）
- **學習獎勵**：完成課程獲得 EDU 獎勵
- **成就系統**：達成學習目標獲取代幣
- **推薦計劃**：邀請新學習者獲得獎勵

### 3. 治理權利
- **課程審核**：投票決定平台上線內容
- **基金分配**：決定教育資助的發放
- **協議升級**：參與平台技術決策

## 代幣經濟學

### 分配方案
| 類別 | 比例 | 說明 |
|------|------|------|
| 生態基金 | 25% | 用戶激勵和合作夥伴 |
| 團隊 | 20% | 長期鎖倉 |
| 早期支持者 | 15% | 種子和私募投資者 |
| 金庫 | 15% | 協議發展儲備 |
| 流動性 | 10% | 交易所和 DeFi |
| 公開銷售 | 10% | 社區分配 |
| 顧問 | 5% | 專家顧問團隊 |

### 解鎖安排
- 團隊代幣：3-4 年線性解鎖
- 投資者代幣：分階段解鎖
- 生態基金：持續釋放支持增長

## 使用場景

### 對學習者
- **支付學費**：使用 EDU 購買課程
- **賺取回饋**：完成學習獲得獎勵
- **建立聲譽**：鏈上學習記錄證明能力

### 對教育者
- **內容變現**：出售課程和教材
- **版稅收入**：每次轉售獲得分成
- **募資工具**：通過 EDU 眾籌新課程

### 對機構
- **企業培訓**：使用 EDU 購買企業課程
- **認證服務**：發放區塊鏈證書
- **人才招聘**：驗證求職者技能

---

## 合作夥伴生態

### 教育平台整合
- **TinyTap**：兒童教育內容平台
- **Hooked**：遊戲化學習應用
- **多家大學**：探索學位上鏈

### 企業採用
- 企業使用 EDU 進行員工培訓
- 教育機構接受 EDU 支付
- 政府探索教育憑證系統

## 創新機制

### Publisher NFT
教育者可以將課程鑄造為 NFT：
- **所有權證明**：明確內容歸屬
- **收益分配**：自動執行版稅
- **融資工具**：出售未來收益權

### 學習證明 (Proof of Learning)
- 不可篡改的學習歷史
- 可驗證的技能證書
- 去中心化身份的一部分
