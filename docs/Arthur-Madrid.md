# Arthur Madrid

**Arthur Madrid** 是 [[The-Sandbox]] 的聯合創辦人兼行政總裁（CEO），負責領導這個領先元宇宙平台的戰略方向和整體發展。他與 [[Sebastien-Borget]] 於2012年共同創立了 Pixowl，後來發展成為今日知名的 The Sandbox 元宇宙。

## 職業經歷

### The Sandbox / Pixowl（2012年至今）

#### 創立與早期發展（2012年-2018年）
Arthur Madrid 於2012年與 [[Sebastien-Borget]] 共同創立 Pixowl，願景是打造一個讓用戶能夠創造、擁有和變現遊戲體驗的平台：

- 開發手機遊戲《The Sandbox》（2012年）
- 建立像素風格創意遊戲社群
- 遊戲累積超過4000萬次下載

#### 區塊鏈轉型與元宇宙（2018年至今）
2018年，[[Animoca-Brands]] 收購 Pixowl，Arthur Madrid 帶領公司進行區塊鏈轉型：
- 將 The Sandbox 發展為3D區塊鏈元宇宙
- 推出 SAND 功能代幣
- 建立 LAND 虛擬土地 NFT 系統
- 打造創作者經濟模式

## 戰略領導

### 元宇宙願景
Arthur Madrid 的戰略願景包括：

| 核心原則 | 說明 |
|----------|------|
| **用戶所有權** | 玩家真正擁有遊戲內資產 |
| **開放生態** | 去中心化的創作者經濟 |
| **互操作性** | 跨平台資產和身份 |
| **社群優先** | 由玩家驅動的內容創作 |

### 業務發展
在 Arthur Madrid 的領導下，The Sandbox 實現了：
- **LAND 銷售額**：超過數億美元的虛擬土地銷售
- **品牌合作**：與超過200個知名品牌建立合作
- **創作者社群**：數百萬註冊用戶和活躍創作者
- **市場地位**：成為領先的元宇宙平台之一
